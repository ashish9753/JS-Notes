üìù String Methods & Concepts
üî§ String Methods Overview

Strings in JavaScript come with many built-in methods to manipulate and work with text data.

let text = "Hello World"; console.log(text.length); // 11 console.log(text.charAt(0)); // "H" console.log(text.indexOf("o")); // 4
‚úÇÔ∏è Trim Method

The trim() method removes whitespace from both ends of a string.

let messy = " Hello World "; console.log(messy.trim()); // "Hello World" console.log(messy.trimStart()); // "Hello World " console.log(messy.trimEnd()); // " Hello World"
üí° Note: trim() doesn't modify the original string - it returns a new one!
üîí Strings are Immutable in JS

Once created, strings cannot be changed. All string methods return new strings.

let original = "Hello"; let modified = original.toUpperCase(); console.log(original); // "Hello" (unchanged) console.log(modified); // "HELLO" (new string)
üîç Key Point: This is different from arrays, which are mutable!
üîÑ ToUpperCase and ToLowerCase

Convert strings between different cases for formatting and comparison.

let text = "JavaScript is Awesome"; console.log(text.toUpperCase()); // "JAVASCRIPT IS AWESOME" console.log(text.toLowerCase()); // "javascript is awesome" // Useful for case-insensitive comparison let input = "HELLO"; if(input.toLowerCase() === "hello") { console.log("Match found!"); }
üîç Methods with Arguments - indexOf

Find the position of a substring within a string.

let sentence = "The quick brown fox jumps over the lazy dog"; console.log(sentence.indexOf("fox")); // 16 console.log(sentence.indexOf("cat")); // -1 (not found) console.log(sentence.indexOf("the")); // 31 (first occurrence) console.log(sentence.lastIndexOf("the")); // 31 (last occurrence)
‚ö†Ô∏è Remember: indexOf() returns -1 if the substring is not found!
üîó Method Chaining

Combine multiple string methods in a single line for powerful text processing.

let userInput = " HELLO WORLD "; let processed = userInput.trim().toLowerCase().replace("world", "javascript"); console.log(processed); // "hello javascript" // Step by step breakdown: // " HELLO WORLD " ‚Üí trim() ‚Üí "HELLO WORLD" // "HELLO WORLD" ‚Üí toLowerCase() ‚Üí "hello world" // "hello world" ‚Üí replace() ‚Üí "hello javascript"
üî™ Slice Method

Extract a portion of a string without modifying the original.

let text = "JavaScript Programming"; console.log(text.slice(0, 10)); // "JavaScript" console.log(text.slice(4)); // "Script Programming" console.log(text.slice(-7)); // "gramming" (from end) console.log(text.slice(-11, -7)); // "Pro" (negative indices)
üìù Syntax: string.slice(startIndex, endIndex) - endIndex is exclusive!
üîÑ Replace & Repeat Method

Replace text and repeat strings for various use cases.

// Replace method let text = "I love cats and cats love me"; console.log(text.replace("cats", "dogs")); // "I love dogs and cats love me" console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs love me" // Repeat method let pattern = "‚≠ê"; console.log(pattern.repeat(5)); // "‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê" console.log("Ha".repeat(3)); // "HaHaHa"

üìä Arrays & Data Structures
üìã Array (Data Structure)

Arrays are ordered collections that can store multiple values of any type.

// Different ways to create arrays let fruits = ["apple", "banana", "orange"]; let numbers = new Array(1, 2, 3, 4, 5); let mixed = ["text", 42, true, null, {name: "John"}]; console.log(fruits.length); // 3 console.log(fruits[0]); // "apple"
üëÅÔ∏è Visualizing Arrays

Understanding array structure and indexing is crucial for manipulation.

let colors = ["red", "green", "blue", "yellow"]; // Index: 0 1 2 3 // Value: "red" "green" "blue" "yellow" console.log(colors[0]); // "red" console.log(colors[3]); // "yellow" console.log(colors[-1]); // undefined (no negative indexing)
üí° Remember: Array indexing starts at 0, not 1!
üèóÔ∏è Creating Arrays

Multiple ways to create and initialize arrays in JavaScript.

// Literal notation (most common) let fruits = ["apple", "banana"]; // Constructor with elements let numbers = new Array(1, 2, 3); // Constructor with length let empty = new Array(5); // Creates array with 5 empty slots // From other iterables let fromString = Array.from("hello"); // ["h", "e", "l", "l", "o"] let range = Array.from({length: 5}, (_, i) => i + 1); // [1, 2, 3, 4, 5]
üîÑ Arrays are Mutable

Unlike strings, arrays can be modified after creation.

let fruits = ["apple", "banana"]; fruits.push("orange"); // Add to end fruits[0] = "grape"; // Change existing element fruits.unshift("mango"); // Add to beginning console.log(fruits); // ["mango", "grape", "banana", "orange"]
üîë Key Difference: Strings are immutable, arrays are mutable!
üõ†Ô∏è Array Methods

Essential methods for array manipulation and transformation.

let numbers = [1, 2, 3, 4, 5]; // Adding/Removing elements numbers.push(6); // Add to end ‚Üí [1, 2, 3, 4, 5, 6] numbers.pop(); // Remove from end ‚Üí [1, 2, 3, 4, 5] numbers.unshift(0); // Add to start ‚Üí [0, 1, 2, 3, 4, 5] numbers.shift(); // Remove from start ‚Üí [1, 2, 3, 4, 5] // Transforming arrays let doubled = numbers.map(n => n * 2); // [2, 4, 6, 8, 10] let evens = numbers.filter(n => n % 2 === 0); // [2, 4] let sum = numbers.reduce((acc, n) => acc + n, 0); // 15

üîç indexOf & includes Method

Search for elements within arrays efficiently.

let fruits = ["apple", "banana", "orange"]; // indexOf - returns index or -1 console.log(fruits.indexOf("banana")); // 1 console.log(fruits.indexOf("grape")); // -1 // includes - returns true/false console.log(fruits.includes("apple")); // true console.log(fruits.includes("grape")); // false // For objects, use find/findIndex let users = [{name: "John"}, {name: "Jane"}]; let user = users.find(u => u.name === "Jane"); // {name: "Jane"}
üîó Concatenation & Reverse

Combine arrays and reverse their order.

let arr1 = [1, 2, 3]; let arr2 = [4, 5, 6]; // Concatenation let combined = arr1.concat(arr2); // [1, 2, 3, 4, 5, 6] let spread = [...arr1, ...arr2]; // [1, 2, 3, 4, 5, 6] // Reverse (mutates original) let original = [1, 2, 3]; let reversed = original.reverse(); // [3, 2, 1] console.log(original); // [3, 2, 1] - original changed! // Non-mutating reverse let safe = [...original].reverse(); // Keep original intact
üî™ Slice in Arrays

Extract portions of arrays without modifying the original.

let numbers = [0, 1, 2, 3, 4, 5]; console.log(numbers.slice(2, 5)); // [2, 3, 4] console.log(numbers.slice(3)); // [3, 4, 5] console.log(numbers.slice(-3)); // [3, 4, 5] console.log(numbers.slice(-5, -2)); // [1, 2, 3] // Original array unchanged console.log(numbers); // [0, 1, 2, 3, 4, 5]
üîß Splice in Arrays

Add, remove, or replace elements at any position (mutates original array).

let fruits = ["apple", "banana", "orange"]; // Remove elements let removed = fruits.splice(1, 1); // Remove 1 element at index 1 console.log(fruits); // ["apple", "orange"] console.log(removed); // ["banana"] // Add elements fruits.splice(1, 0, "grape", "kiwi"); // Add at index 1, remove 0 console.log(fruits); // ["apple", "grape", "kiwi", "orange"] // Replace elements fruits.splice(2, 1, "mango"); // Replace 1 element at index 2 console.log(fruits); // ["apple", "grape", "mango", "orange"]
üìä Sort in Arrays

Arrange array elements in ascending or descending order.

// String sorting let fruits = ["banana", "apple", "orange"]; fruits.sort(); // ["apple", "banana", "orange"] // Number sorting (requires compare function) let numbers = [10, 5, 20, 3]; numbers.sort(); // ["10", "20", "3", "5"] - Wrong! numbers.sort((a, b) => a - b); // [3, 5, 10, 20] - Correct! // Descending order numbers.sort((a, b) => b - a); // [20, 10, 5, 3] // Object sorting let users = [{age: 30}, {age: 25}, {age: 35}]; users.sort((a, b) => a.age - b.age); // Sort by age ascending

üìé Array References

Understanding how arrays are stored and referenced in memory.

// Arrays are reference types let arr1 = [1, 2, 3]; let arr2 = arr1; // Same reference arr2.push(4); console.log(arr1); // [1, 2, 3, 4] - Original changed! // Creating independent copies let original = [1, 2, 3]; let shallow = [...original]; // Shallow copy let deep = JSON.parse(JSON.stringify(original)); // Deep copy (for simple cases)
‚ö†Ô∏è Important: Assignment copies the reference, not the array!
üîí Constant Arrays

const with arrays prevents reassignment but allows mutation.

const fruits = ["apple", "banana"]; // This is allowed (mutation) fruits.push("orange"); // ‚úÖ Works fruits[0] = "grape"; // ‚úÖ Works // This is NOT allowed (reassignment) fruits = ["new", "array"]; // ‚ùå Error! // To make truly immutable const immutable = Object.freeze(["a", "b", "c"]); immutable.push("d"); // ‚ùå Error in strict mode
üèóÔ∏è Nested Arrays

Arrays can contain other arrays, creating multi-dimensional structures.

// 2D Array (Matrix) let matrix = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]; console.log(matrix[0][1]); // 2 console.log(matrix[2][0]); // 7 // Jagged arrays (different lengths) let jagged = [ [1, 2], [3, 4, 5, 6], [7] ]; // Flattening nested arrays let nested = [[1, 2], [3, 4], [5, 6]]; let flat = nested.flat(); // [1, 2, 3, 4, 5, 6] let deepFlat = [[[1]], [[2]]].flat(2); // [1, 2]
